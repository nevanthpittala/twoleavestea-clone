# Two Leaves Tea Clone

A premium tea website clone inspired by Two Leaves and a Bud.

## Live Demo

https://twoleaves-tea.vercel.app

## Built With

- HTML
- CSS
- Swiper.js

## Features

- Responsive layout
- Product sliders
- Reviews section
- Latte showcase section
- Tea journal section
- Premium UI design
## Run Locally

Open `index.html` in your browser.
